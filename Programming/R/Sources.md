# Here you'd find all the sources for coding with R

> R https://cran.r-project.org/

R is a free software environment for statistical computing and graphics. It compiles and runs on a wide variety of UNIX platforms, Windows and MacOS. To download R, please choose your preferred CRAN mirror.


> R studio https://www.rstudio.com/ 

An integrated development environment for R and Python, with a console, syntax-highlighting editor that supports direct code execution, and tools for plotting, history, debugging and workspace management.


> Visual studio code https://code.visualstudio.com/

Visual studio code is an excellent code editor for both begginers and experts alike, it's highly advised that you use this code editor for writing your programming sessions.
